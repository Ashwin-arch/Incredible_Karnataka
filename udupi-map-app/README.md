# Udupi Map App

A modern, responsive web application for visualizing important places in Udupi. Built with React, Vite, Leaflet, and Tailwind CSS.

## Features

- Interactive map with markers for key locations in Udupi.
- Beautiful, responsive UI using Tailwind CSS.
- Fast development and build process with Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

## Installation

1. Clone the repository (if you haven't already):
   ```bash
   git clone https://github.com/Ashwin-arch/Incredible_Karnataka/
   cd udupi-map-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Getting Started

### Development

To run the app in development mode with Hot Module Replacement (HMR):

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Production

To build the app for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

To preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **React**: UI library
- **Vite**: Build tool and dev server
- **Leaflet & React-Leaflet**: Map visualization
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Icon set
